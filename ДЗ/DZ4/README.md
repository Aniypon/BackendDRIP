# DZ4: JDBC примеры, воспроизведенные в JPA

В проекте реализованы JPA-эквиваленты JDBC-сценариев из семинара:
- insertUser
- insertMultipleUsers
- insertWithTransaction
- insertWithTransactionRollback
- testReadNotCommitted
- notDirtyRead
- dirtyRead
- repeatableRead
- notRepeatableRead
- phantomRead
- notPhantomRead
- anomalyExample
lyExample
- propagationRequired
- propagationSupports
- propagationRequiresNew

## Запуск
```bash
docker compose up --build app
```
